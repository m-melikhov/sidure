# 🕍 Siddur — Еврейский молитвенник

## Требования
- Node.js 18+ (скачать: https://nodejs.org)

## Запуск

1. Открой терминал в этой папке
2. Установи зависимости (один раз):
   ```
   npm install
   ```
3. Запусти приложение:
   ```
   npm run dev
   ```
4. Открой браузер: http://localhost:5173

## Сборка для продакшена
```
npm run build
```
Готовые файлы окажутся в папке `dist/` — их можно открыть с любого хостинга.

---
Приложение работает и на телефоне: просто открой http://[IP-твоего-компьютера]:5173 с телефона в той же сети.
